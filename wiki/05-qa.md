## 已知问题

### BUG-001：筛选功能未实现
- **位置：** `ClipLibrary.tsx` 中的三个 Select 组件
- **现象：** 切换 SOURCE KOL / CATEGORY / SORT 下拉框，片段列表不变化
- **原因：** Select 的 `defaultValue` 固定，未绑定 state，未实现过滤逻辑
- **优先级：** 中
- **修复方案：**
  ```tsx
  // 添加 state
  const [selectedKol, setSelectedKol] = useState('all');
  const [selectedCategory, setSelectedCategory] = useState('all');
  const [sortBy, setSortBy] = useState('newest');

  // 计算过滤后的 clips
  const filteredClips = useMemo(() => {
    let result = clips;

    if (selectedKol !== 'all') {
      result = result.filter(c => c.kolName === selectedKol);
    }

    if (selectedCategory !== 'all') {
      result = result.filter(c => c.topicCategory === selectedCategory);
    }

    if (sortBy === 'newest') {
      result = [...result].sort((a, b) => new Date(b.createdAt).getTime() - new Date(a.createdAt).getTime());
    } else {
      result = [...result].sort((a, b) => new Date(a.createdAt).getTime() - new Date(b.createdAt).getTime());
    }

    return result;
  }, [clips, selectedKol, selectedCategory, sortBy]);
  ```

### BUG-003：搜索结果为硬编码
- **位置：** `Search.tsx`
- **现象：** 无论输入什么查询词，结果始终相同（`clips.filter(c => c.relevance)` 只过滤有 relevance 字段的 clip）
- **原因：** 未接入真实语义搜索 API
- **优先级：** 高
- **修复方案：**
  ```tsx
  // 前端临时方案（关键词匹配）
  const handleSearch = async (query: string) => {
    setIsSearching(true);

    // 模拟网络延迟
    await new Promise(resolve => setTimeout(resolve, 800));

    // 关键词匹配
    const results = clips
      .map(clip => {
        const titleMatch = clip.title.toLowerCase().includes(query.toLowerCase());
        const summaryMatch = clip.summary.toLowerCase().includes(query.toLowerCase());
        const keywordMatch = clip.keywords.some(k => k.toLowerCase().includes(query.toLowerCase()));

        let relevance = 0;
        if (titleMatch) relevance += 30;
        if (summaryMatch) relevance += 20;
        if (keywordMatch) relevance += 10;

        return { ...clip, relevance };
      })
      .filter(c => c.relevance > 0)
      .sort((a, b) => b.relevance - a.relevance);

    setSearchResults(results);
    setIsSearching(false);
    setHasSearched(true);
  };

  // 后端方案（Phase 3 实现）
  const response = await fetch('/api/clips/search', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ query })
  });
  const data = await response.json();
  setSearchResults(data.results);
  ```

### BUG-004：合成页时间轴初始化固定
- **位置：** `Combine.tsx` 第 9 行
- **现象：** 时间轴始终加载前 5 个 clip，无法从搜索结果或片段库添加
- **原因：** `useState<Clip[]>(clips.slice(0, 5))` 硬编码
- **优先级：** 中
- **修复方案：**
  ```tsx
  // 从 store 获取 timeline
  const { timeline, setTimeline } = useAppStore();

  // 在 store.ts 中添加
  interface AppState {
    timeline: Clip[];
    setTimeline: (clips: Clip[]) => void;
  }

  // Search 页面 Lucky Combo 完成后
  const selectedClips = await fetch('/api/lucky-combo', { ... });
  setTimeline(selectedClips);
  setActivePage('combine');

  // ClipLibrary 页面添加到时间轴
  const addToTimeline = (clip: Clip) => {
    setTimeline([...timeline, clip]);
  };
  ```
- **状态：** ⏳ 待实现（Phase 6）

### BUG-005：任务进度不实时更新
- **位置：** `TaskMonitor.tsx` + `store.ts`
- **现象：** running 任务的进度条固定在初始值（62%），不会自动推进
- **原因：** 无 WebSocket 或轮询机制，store 中无进度更新 action
- **优先级：** 高
- **修复方案：**
  ```tsx
  // 前端轮询方案
  useEffect(() => {
    const interval = setInterval(async () => {
      const response = await fetch('/api/jobs?status=running');
      const data = await response.json();
      setJobs(data.jobs);
    }, 2000); // 每 2 秒轮询一次

    return () => clearInterval(interval);
  }, []);

  // 后端 WebSocket 方案（Phase 2 实现）
  const ws = new WebSocket('ws://localhost:3001/ws');
  ws.onmessage = (event) => {
    const job = JSON.parse(event.data);
    updateJobProgress(job.id, job.progress);
  };
  ```
